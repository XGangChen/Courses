Content

In this assignment, we are trying to implement a magic trick where we read the user’s mind via the Internet or using our humanoid robot Thormang during a life show.

The trick works as follows:

  1. The avatar/robot introduces the human volunteer to the four magic symbols: triangle, wave, square, and star.
  2. The robot then instructs the human to prepare a small notepad and a pen and to draw the shape while holding the notebook up to the camera. The first part of the project is to collect several training and testing videos. Some videos are available in  Magic Sign Drawing. In this video, I draw the four symbols, one after another.

User drawing a secret symbol on the notepad.

  1. The trick works because you can usually determine the symbol that was drawn by the user by carefully watching the hand motions of the user.
  2. The first part of the project is thus to implement a robot vision system that can determine the sign drawn by looking at the motion. There are several possibilities for your algorithm such as edge/contour detection, optical flow, HOG features, or deep learning algorithms such as Google’s mediapipe.
  3. Magicians and psychologists have known for a long time that people are much more likely to choose the star than the other symbols. One part of our system will be to verify this result.
  4. The robot then pretends to read the mind of the human and presents a prediction.
  5. The user shows the drawn symbol to the robot. Hence, the third part of the project is to implement an image recognition system that can detect hand drawn magic signs.
  6. In the second part of the assignment, the user will show the drawn shape to the camera and your program should recognize the shape.
  7. In the third part of the assignment, you want the use to draw the shape on the desk rather than in the air, because this is more natural.

Submission

All your files and analysis should go into a directory labeled <user_id>_CV_A1, where <user_id> is your own user id.

Add a README.txt file into the directory, which shows: (a) what other sources you used apart from the lecture material used in class during your work on the assignment, and (b) how to compile and run your program, and (c) any interesting features and extensions of your assignment.

Add a file evaluation.pdf, which shows the results of the requested evaluations. Discuss the results. Were the results expected or did they surprise you? Did the results highlight shortcomings in the system? Do you have any ideas for how to fix those?

Submit your assignment via email to jacky.baltes@ntnu.edu.tw. You can submit a compressed directory of your source code and analysis directly if the resulting file is less than 2 Megabyte. If the file is larger than create a google drive folder and send me a sharable link to the folder to my email.
